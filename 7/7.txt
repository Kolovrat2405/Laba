8. Программирование
```
#include <iostream>
int main() {
    int a;
    double b;
    a = 5;
    b = 5;
    std::cout << a << b;
    return 0;
}
```
-какое значение содержит переменная a в 5й строке?

a=2310523

Какое значение содержит переменная b в 6й строке?

b=1.20452844391e-322

Какие значение содержатся в переменных a и b в 7й строке?

a=5

b=5
